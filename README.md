# SOC-Analyst-HandsOn-Repository
Overview
This repository documents my hands-on learning journey to become a SOC Analyst.
It focuses on practical SOC workflows such as log analysis, SIEM alert handling,
threat detection, and incident response.

Topics Covered
- SOC Fundamentals
- Log Analysis (Windows & Linux)
- SIEM Basics (Splunk)
- Threat Detection Techniques
- Incident Response Lifecycle

Tools & Technologies
- Windows Event Viewer
- Linux (auth logs)
- Splunk (Basic)
- Wireshark
- Nmap
- VirusTotal

Goal
To demonstrate practical SOC knowledge and readiness for a Tier-1 SOC Analyst role.
